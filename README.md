# خطب الغزالي — Ghazali's Sermons

Interactive trilingual web app based on **"Khutab al-Ghazali fi Shu'un al-Din wal-Hayat"** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 sermon excerpts on faith, society, justice, worship, family, and knowledge
- 6 major theme categories
- Interactive quiz
- Narrator with karaoke, speed/pitch control, sleep timer
- Three themes (Teal, Night, Sage)
- Three languages (Arabic, English, French)
- Dua panel and daily sermon

## Tech

Pure HTML/CSS/JS — no frameworks, no build step. Works offline as a PWA.
